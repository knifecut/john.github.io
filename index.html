APLICACIÓN WEB PARA EL SEGUIMIENTO DEL APRENDIZAJE ACTIVO EN ASIGNATURAS DE ESTUDIOS UNIVERSITARIOS

Noviembre 2018 - Junio 2019

Universidad de Jaén

Desarrollo de una aplicación web con el objetivo de realizar un seguimiento individualizado del uso de técnicas de aprendizaje activo por parte de los estudiantes que, voluntariamente, deseen participar en dicho programa en la asignatura Programación y Administración de Redes.

La aplicación ofrecerá dos perfiles diferentes: estudiante y profesor. El primero podrá acceder a un conjunto de recursos específicos, de cuyo uso se efectuará el seguimiento. Al segundo se ofrecerá información tanto detallada como agregada, numérica y visualmente, de dicho uso por parte de los estudiantes participantes.

Tecnologías: HTML5 | CSS3 | JavaScript | Ajax | MySQL | Bootstrap
